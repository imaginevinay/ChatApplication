Events to listen

verifyUser event
This event ("verifyUser") has to be listened on the user's end to verify user authentication.
user will only be set as online user after verification of authentication token.
online-user-list event
This event ("online-user-list") has to be listened on the user's end to identify all available users that are currently online.
The output will be an object, object has key as userId and value as full name.
setJoinedRoom
This event ("setJoinedRoom") has to be listened on the user's end to get teh data
who ahs joined a room like roomId,roomName and joined User name.This event is broadcast to everyone who has joined a room.
The output will be an object, object has keys roomId,roomName,joinedUser with its values.
receiveMsg event
This event ("receiveMsg") has to be listened to get all chats of that Room.The chat which is sent by user will
be listened to each member who has joined a room.
Example output it as follows:
                            {
                                chatId: 'unique chat id',
                                message: 'you message',
                                createdOn: date, 
                                receiverId: 'chatRoomId',
                                receiverName: 'chatRoomName',
                                senderId: 'userId of sender',
                                senderName: 'full name of sender'
                            }
userTyping event
This event ("userTyping") has to be listened to get data who is typing.This data will be emitted to every
member of Room.

Events to emit

setUser event
This event ("setUser") has to be emitted when a user comes online.
User can only be set as online only after verification of authentication token. Which you have pass here. The following data has to be emitted:
authentication token
joinRoom event
This event ("joinRoom") has to be emitted when a user wants to join a room.
The following data has to be emitted for joining a specific room:
Room Id
Room Name
The name of the person who wants to join.
room-msg event
This event ("room-msg") has to be emitted to identify an chat message that has been send to room.
The following data has to be emitted:
                            {
                                senderName: 'John Doe',
                                senderId: 'userId of sender',
                                receiverName: chatRoomName',
                                receiverId: 'chatRoomId of Room',
                                message: 'your chat message',
                                createdOn: date
                            }
typing
This event ("typing") has to be emitted, when the user start typing.
The following data has to be emitted
                           {
                                                 typingUser:'User who is typing'
						typingUserId:'UserId of typing user',
						roomId:'ChatRoomId'
			               }
disconnection
This event ("disconnection") has to be emitted, when the user gets disconnected(logout).
